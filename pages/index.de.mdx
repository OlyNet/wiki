fSo wechseln Sie von einer statischen IP-Adresse zu DHCP:
– Schließen Sie zunächst Ihren Router an und verbinden Sie ihn (auch wenn er keine Internetverbindung hat).
– Öffnen Sie in Ihrem bevorzugten Webbrowser die Systemsteuerung Ihres Routers. Die Adresse lautet in der Regel 192.168.01 oder 192.168.1.1 (bei TP-Link), kann aber je nach Hersteller abweichen.
– Geben Sie Ihre Zugangsdaten ein (falls Sie diese vergessen haben, müssen Sie Ihren Router zurücksetzen).
– Gehen Sie zu WAN/Interneteinstellungen -> Wählen Sie DHCP (Dynamische IP-Adresse).
– Entfernen Sie ggf. manuell eingegebene IP-Adressen, Subnetze, Gateways oder DNS-Einstellungen.
– Speichern Sie die Änderungen. Fertig!

So setzen Sie Ihren Router zurück:

– Suchen Sie die Reset-Taste: Auf der Rückseite Ihres Routers, in der Nähe der Anschlüsse, befindet sich eine Reset-Taste. Manchmal befindet sich in der Öffnung ein kleines Loch, in das Sie einen spitzen Gegenstand wie einen Stift oder eine Tackernadel stecken müssen.
- Führen Sie die Klammer ein oder halten Sie die Taste 10 Sekunden lang gedrückt. Falls Ihr Router über LEDs verfügt, blinken diese möglicherweise.
- Anschließend wird das Gerät auf die Werkseinstellungen zurückgesetzt. Die Standard-WLAN-Anmeldedaten (SSID und Passwort) finden Sie auf der Rückseite des Geräts.
- Sie können nun zur vorherigen Anleitung zurückkehren, um die Einrichtung als DHCP fortzusetzen.
