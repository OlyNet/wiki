# How to config your new TP-Link TL-WR940N Router

### Things you need:

- Netzdaten-Page (if you don't have one contact: [wohnen-team1@stwm.de](mailto:wohnen-team1@stwm.de))
  <Image
    src="/assets/tl-wr940/6.png"
    alt="Picture 6"
    width={300}
    height={200}
  />
- TP-Link TL-WR940N Router

### Setup

1. If your router isn't new perform a reset as shown in the manual
2. Connect to your router

   2.1. Wired: Using one of the yellow slots (Picture 1) and connect it to your computer.

   {' '}

   <Image
     src="/assets/tl-wr940/1.png"
     alt="Picture 1"
     width={300}
     height={200}
   />

   2.2. Wireless: Using the credentials written on the bottom of the router (Picture 2)

   <Image
     src="/assets/tl-wr940/2.png"
     alt="Picture 2"
     width={300}
     height={200}
   />

3. After connecting to the router, enter the interface of the router using a webbrowser (e.g. Safari, Chrome, etc.) using [http://tplinkwifi.net](http://tplinkwifi.net/) (Picture 3 and 4)
   <Image
     src="/assets/tl-wr940/3.png"
     alt="Picture 3"
     width={300}
     height={200}
   />
   <Image
     src="/assets/tl-wr940/4.png"
     alt="Picture 4"
     width={300}
     height={200}
   />
4. Create a profile
5. Click on Network -> WAN -> Select „Static IP" -> Enter the IP-addresses from the Netzdaten-Page -> Click Save
   {' '}
   <Image
     src="/assets/tl-wr940/7.png"
     alt="Picture 7"
     width={400}
     height={300}
   />
6. After configuration connect an ethernet cable with the blue slot of the router and the other end with the wall box , where it says „EDV" (Picture 1.1)
   <Image
     src="/assets/tl-wr940/11.png"
     alt="Picture 1.1"
     width={300}
     height={200}
   />
7. Now you should have Internet.
